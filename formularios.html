Los formularios y sus etiquetas

En esta sección se presentarán todas las etiquetas necesarias junto con sus atributos para crear un buen formulario

La etiqueta <button>

Define un botón dentro de un formulario

Atributos de la etiqueta <button>

autofocus="booleano"
Indica si el elemento va a recibir el foco en el momento de cargar la página
disabled="booleano"
Indica si el elemento esta desactivado
form="nombre"
Nombre del formulario asociado
formaction="URL"
URL que se disparará al activar el elemento
formenctype="modo"
Modo de envío de los datos
formmethod="método"
Método HTTP de la acción
formnovalidate="booleano"
Indica si no se va a validar el formulario antes de enviarse
formtarget="target"
Ventana destino
name="nombre"
Nombre del botón
type="tipo"
Tipo del botón ("button", "reset", "submit"...)
value="valor"
Valor que devuelve el control (en algunos tipos)

La Etiqueta <datalist>

La etiqueta <datalist> se utiliza para enumerar un conjunto de datos que se utilizan en una lista de opciones como en los formularios. Esta característica tiene que ser utilizada junto con el elemento imput (como se ve en el ejemplo).

Ejemplo:

<html>
  <body>
    <input list="Cellphones" />
      <datalist id=" Cellphones ">
        <option value="BENQ">
        <option value="BLACKBERRY">
        <option value="ERICCSSON">
        <option value="FLY">
        <option value="HTC">
      </datalist>
  </body>
</html>

La Etiqueta <fieldset>

Permite agrupar lógicamente varios campos y etiquetas de un formulario

Atributos de la etiqueta <fieldset>

disabled="booleano"
si el elemento está desactivado
form="nombre"
Nombre del formulario asociado
name= "nombre"
Nombre de la agrupación

La Etiqueta <form>

Abre una sección de formulario

Atributos de la etiqueta <form>

accept-charset="codificación"
Codificación de caracteres aceptada
action="URL"
URL del programa que se ejecutará para procesar el formulario
autocomplete="on|off"
Indica si está activada o desactivada la función de autocompletar
enctype="modo"
Modo empleado en la codificación del contenido
method="método"
Método HTTP de la acción
name="nombre"
Nombre simbólicodel formulario
novalidate="booleano"
Indica si no se va a validar el formulario antes de enviarse
formtarget="target"
Ventana destino

La Etiqueta <imput>

Control de entrada

Atributos de la etiqueta <imput>

accept="lista de tipos"
Lista de tipos aceptables en caso de una entrada de tipo "file"
alt="texto"
Texto alternativo que se mostrará si el elemento es una imagen y ésta no se puede descargar
autocomplete="on|off"
Indica si está activada o desactivada la función de autocompletar
checked
Indica si el elemento está o no inicialmente seleccionado (algunos tipos)
disabled="booleano"
Indica si el elemento está desactivado
form="nombre"
Nombre del formulario asociado
formaction="URL"
URL que se disparará al activar el elemento
formenctype="modo"
Modo de envío de los datos
formmethod="método"
Método HTTP de la acción
formnovalidate="booleano"
Indica si no se va a validar el formulario antes de enviarse
formtarget="target"
Ventana destino
height="alto"
Altura del elemento
list="identificador"
Identificador de una lista definida con <datalist> q contiene las opciones
max="máximo"
Valor máximo
maxlength="ancho"
Longitud máxima de la infromación
min="mínimo"
Valor mínimo
multiple="booleano"
Indica si es posible introducir múltiples valores
name="nombre"
Nombre de la variable asociada al elemento
pattern="expresión"
Expresión regular para validar el contenido de la entrada
placeholder="texto"
Texto que se usa como guía para que el usuario introduzca información
readonly="booleano"
Indica si es posible cambiar el contenido de la entrada
required="booleano"
Indica si es obligatorio rellenar el contenido de la entrada
size="caracteres"
Anchura de la ventana del control en caracteres
src="URL"
Cuando el tipo es imágen especifica la dirección donde debe abrirla
step="paso"
Permite definir el númerode posibles tramos en un valor numérico
type="tipo"
Tipo del control (text, reset,radiobutton,submit...)
value="valor"
Valor que devuelve el control
height="ancho"
Ancho del elemento

La Etiqueta <keygen>

Genera clave para transferir de manera segura la información. El atributo keytype indica el método usado para el cifrado ("rsa", "dsa"...)

Ejemplo:

<html>
  <body>
    <keygen name="name" challenge="challenge string" keytype="type" keyparams="pqg-params">
  </body>
</html>

La Etiqueta <label>

Delimita una etiqueta para una entrada del tipo <imput>

La Etiqueta <legend>

Permite asignar un título a una agrupación de campos y etiquetas (<fieldset>)

La Etiqueta <optgroup>

Agrupa opciones para una entrada de tipo <select>. El atributo "label" es la etiqueta de cada una de las agrupaciones de <option>

La Etiqueta <option>

Componente de una lista desplegable o de selección

Atributos de la etiqueta <option>

disabled="booleano"
Indica si el elemento está desactivado
label="etiqueta"
Etiqueta de la opción
selected
Indica que la opción está seleccionada inicialmente
value="valor"
Valor devuelto por la opción

La Etiqueta <output>

Muestra el resultado de un cálculo. El atributo for indica a qué campos de entrada hacer referencia

La Etiqueta <select>

Lista desplegable

Atributos de la etiqueta <select>

autofocus="booleano"
Indica si el elemento va a recibir el foco de atención en el momento de cargar la página
disabled="booleano"
Indica si el elemento está o no desactivado
form="nombre"
Nombre del formulario asociado
multiple
Permite seleccionar varios elementos.Fuerza la aplicación de una lista de selección
name="texto"
Nombre de la variable
size="número"
Amplitud de la ventana de visualización en líneas. Si el tamaño es mayor que 1 se fuerza una lista de selección

La Etiqueta <textarea>

Entrada en forma de bloque de texto. El valor por defecto es el texto contenido entre esta etiqueta y su cierre

Atributos de la etiqueta <textarea>

autofocus="booleano"
Indica si el elemento va a recibir el foco al cargar la página
cols="número"
Número de columnas del texto
disabled="booleano"
Indica si el elemento está desactivado
form="nombre"
Nombre del formulario asociado
maxlength="ancho"
Longitud máxima de la información
name="texto"
Nombre de la variable
placeholder="texto"
Texto que se utiliza como guía para que el usuario introduzca la información
readonly
Indica que el campo es solo de lectura
required="booleano"
Indica si es obligatorio rellenar el contenido de la entrada
rows="número"
Número de filas del texto
wrap="tipo"
Define la forma en la que se dividirán las palabras al final de la linea (off, soft, hard...)
