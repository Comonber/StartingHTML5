¿Y Ahora Qué?

La Etiqueta <script>

Delimita el código fuente de un script.

Atributos:

charset="codificación"
Juego de caracteres que se va a usar en el script
async="booleano"
Indica si el script debe ejecutarse de inmediato
defer="booleano"
Indica si el script debe ejecutarse cuando termine de cargarse la página
src="URL"
Localización de un script externo
type="lenguaje"
Lenguaje usado en el script
